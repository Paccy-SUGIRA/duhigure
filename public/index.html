About the Program & System Guide

DUHIGURE MU MIRYANGO is a family-based performance contract system aimed at developing communities in Kibungo sector. Families set shared goals (imihigo) and individual duties to achieve collective progress. The system is scalable to NGOMA District and beyond.

Guide: Register your family, add members with details, assign duties. Track progress via the dashboard. Assisted entry available via Intore mu Ikoranabuhanga.
