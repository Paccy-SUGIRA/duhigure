Family Performance Contract System – Kibungo Sector

About the Program

DUHIGURE MU MIRYANGO is a family-based performance contract system designed to improve socio-economic development at household level.

  • Each family is registered with all members
  • Family goals (Imihigo) are defined and monitored
  • Each member has individual performance duties
  • Authorities and Digital Youth (Intore mu Ikoranabuhanga) can assist families

User Guide

  1. Register your family information
  2. Add all family members
  3. Define family goals (Imihigo)
  4. Assign individual responsibilities
  5. Track progress through the dashboard
